================PBR Modular Reading Room===============
Requires Unity 5.5.0 or higher.



Ready for VR.

This pack includes 50 modular prefabs for creating reading rooms, vintage halls, other big fantasy interior areas.

PBR Textures - 4K* (albedo, normal, occlusion, smoothness, height), some 2K, some 1K.

Day and Night Demo scenes included.

* Note: max texture size should be adjusted in texture import settings (up to 4096 per texture). Default package size is 2048 for fast import.
